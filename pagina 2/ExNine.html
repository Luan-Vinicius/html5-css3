CRIANDO LISTAS

Como ja diz o título. Podemos também criar listas usando duas Tags diferentes:
→ <OL> (Ordered Lists), ou Listas Ordenadas em portugues.
→ <UL> (Unordered Lists), ou Listas não Ordenadas em portugues

Usamos a Tag <Li> (List item), para criar os itens da lista

A tag <ol> possui um parâmetro type, onde configuramos o tipo de marcador da lista atual.
As opções para esse parâmetro são:
→ 1 - valor padrão, cria listas numeradas. ex.: 1, 2, 3, 4..
→ A - cria listas alfabéticas em Maiúsculas. ex.: A, B, C, D..
→ a - cria listas alfabéticas em Minúsculas. ex.: a, b, c, d..
→ I - cria listas com algarismo romano em Maiúsculas. ex.: I, II, III, IV...
→ i - cria listas com algarismo romano em Minúsculas. ex.: i, ii, iii, iv...

A tag <ul> possui um parâmetro type, onde configuramos o tipo de marcador da lista atual.
As opções para esse parâmetro são:
→ disc - formato padrão, cria uma lista com uma bolinha preenchido em preto;

  • item 1

→ circle - cria uma lista com uma bolinha sem preenchimento;

  • item 1

→ square - cria uma lista com um quadrado preenchido em preto;

Este tipo de lista é utilizada com a tag <dl> (definition list). Com ela criamos uma lista de termos, com suas respectivas descrições. Como um dicionário.

Utilizamos a tag <dt> (definition term) para cada termo;

Utilizamos a tag <dd> (definition description) para a descrição de cada termo
