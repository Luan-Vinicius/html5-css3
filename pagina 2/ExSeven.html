O que é Semantica?

Semantica é aquilo que tem significado.
No HTML4, muitas tags davam valor a forma do texto do que o significado
Agora no HTML5, a linguagem tenta dar prioridade ao significado das palavras (a semantica), do que a forma (o visual)
O trabalho do visual das palavras. A forma. A linguagem CSS foi encarregada dessa tarefa.
Portanto, na linguagem atual do HTML, precisamos saber separar a semantica/significado, da forma/visual
